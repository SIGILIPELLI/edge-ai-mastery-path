# Edge AI Mastery Path

A free, structured, module-wise Edge AI & TinyML training program — entry
level to master level, covering the full on-device machine-learning
workflow: train tiny models with Keras, convert and quantize them with
TensorFlow Lite (LiteRT), and run them with TFLite-Micro on
microcontrollers. Every Level 1 module runs on a normal laptop first —
no hardware required to start — with microcontroller deployment shown as
the follow-on step.

**Live site:** https://sigilipelli.github.io/edge-ai-mastery-path/

## Contents

- **Level 1 · Entry** — what edge AI is, the train→convert→quantize→deploy workflow, training a tiny Keras model, TFLite conversion, int8 quantization, TFLite-Micro & C arrays, deploying to ESP32/Arduino, sensors & feature extraction, evaluation & debugging, capstone (gesture recognition)
- **Level 2 · Intermediate** — audio wake words, image classification on MCUs, Edge Impulse workflows, dataset design, pruning & distillation, quantization-aware training, ESP32-CAM projects, person detection, memory/latency optimization
- **Level 3 · Advanced** — edge NPUs & accelerators (Coral Edge TPU, Ethos-U), vendor toolchains, ONNX Runtime, streaming inference, object detection, ML compiler stacks, model security, sensor fusion, MLPerf Tiny benchmarking
- **Level 4 · Master** — production edge AI architecture, OTA model updates, fleet monitoring & drift, on-device learning, federated learning, custom kernels & CMSIS-NN, ultra-low-power inference, hardware co-design, privacy & compliance
